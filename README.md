# Algoritmos e Programacao em Python

Repositorio com exercicios, praticas das minhas aulas de Algoritmos e Programacao em Python no 1o semestre.

Aqui voce vai encontrar codigos simples e progressivos, feitos durante o processo de aprendizado, cobrindo desde estruturas condicionais ate repeticoes e funcoes.

## Conteudo

- [aula 03-selecaomultipla.py](/home/vinicius/projetos-universidade/ALGPROG_1sem/aula%2003-selecaomultipla.py): exercicios com `if`, `elif` e `else`
- [aula 04-introducao a while.py](/home/vinicius/projetos-universidade/ALGPROG_1sem/aula%2004-introducao%20a%20while.py): praticas iniciais com `while`
- [aula 05-matchcase-repeticao.py](/home/vinicius/projetos-universidade/ALGPROG_1sem/aula%2005-matchcase-repeticao.py): exercicios com `match case` e repeticao
- [aula 06-introducao a for-range.py](/home/vinicius/projetos-universidade/ALGPROG_1sem/aula%2006-introducao%20a%20for-range.py): exercicios com `for` e `range`
- [aula 07-introducao a funções.py](/home/vinicius/projetos-universidade/ALGPROG_1sem/aula%2007-introducao%20a%20fun%C3%A7%C3%B5es.py): primeiros exercicios com funcoes, parametros e `return`

## O que esta sendo praticado

- Estruturas condicionais
- Operadores logicos e aritmeticos
- Lacos de repeticao com `while` e `for`
- Leitura de dados com `input()`
- Funcoes em Python
- Resolucao de problemas passo a passo

## Objetivo do repositorio

Este repositorio foi criado para registrar minha evolucao nos estudos de logica de programacao e Python.

A ideia nao é reunir codigos "perfeitos", mas mostrar o processo de aprendizado, os exercicios resolvidos em aula e a pratica constante dos fundamentos.

## Como executar

Tenha o Python 3 instalado e rode qualquer arquivo com:

```bash
python nome-do-arquivo.py
```

Exemplo:

```bash
python "aula 07-introducao a funções.py"
```

## Observacoes

- Os arquivos seguem a ordem das aulas
- Os codigos foram mantidos de forma simples, acompanhando o nivel introdutorio da disciplina
